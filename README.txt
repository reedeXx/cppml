
Práctica 2 : CPPML 

cppml es un lenguaje de marcado que permite generar codigo de c++ para declarar librerias , namespace , definir matrices, bucles y estructuras condicionales sencillas empleando la siguiente sintaxis.

Matrices de n dimesiones:

	%VECTOR tipo(DECIMAL NATURAL BOLEANO) N: nombre tam 3%

Librerias:

	LYBRARY% nombrelibreria %

Namespace:

	NAMESPACE% nombre del namespace %

Bucles:

	%LOOP FOR C: valor condicion incremento B: contenido %
	%LOOP WHILE C: condicion B: contenido del bucle %

Sentencias condicionales:

	%IF condicion B:  contenido % 
	%ELSE  contenido ;
